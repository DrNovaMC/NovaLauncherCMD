<h2 align="center"> 📖 Introducción </h2>

**NovaLauncherCMD** es un launcher desarrollado en **Batch (.bat)** para Windows.

Nació como una herramienta para administrar mis propios proyectos, pero también puede utilizarse como una plantilla para desarrolladores que buscan crear lanzadores simples, rápidos y fáciles de personalizar.

<h2 align="center"> ✨ Características </h2>

- ⚡ Ligero y rápido
- 🔧 Fácil de modificar
- 📂 Ideal para proyectos personales
- 📦 Distribución sencilla

---

<h2 align="center"> 🔧 Instalación </h2>

1. Descargue la última versión de **NovaLauncherCMD**.
2. Extraiga el archivo `.zip` en la ubicación que desee.
3. Abra la carpeta extraída.
4. Ejecute **`NovaLauncher.bat`**.

> **Nota:** No requiere instalación. Basta con extraer el contenido y ejecutar el archivo.

---

<h2 align="center"> ⚠️ Compatibilidad </h2>

NovaLauncherCMD es compatible con:

- ✅ Windows 10
- ✅ Windows 11

Las versiones anteriores (Windows 8.1, Windows 8, Windows 7 y más antiguas) **no son compatibles** debido a limitaciones del sistema y del intérprete de comandos de Windows.
